TypeScript def :

TypeScript est un language de programmation Open Source dévéloppé par Microsoft. Il est conçu comme un sur-ensemble typé de JavaScript, ce qui signifie que tout code JavaScript valide est également du TypeScript valide. Cependant, TypeScript offre des fonctionnalité supplémentaires qui améliorent le dévéloppement et la maintenance des applications JavaScript à grande échelle.

Les principales fonctionnalités

1. Typage TypeScript permet de définir des types pour les variables, les paramètres de fonction, les valeurs de retour et d'autres éléments du code. Cela permet de détecter les erreurs de typage lors de la phase de dévéloppement plutôt qu'à l'exécution, ce qui peut améliorer la fiabilité du code.

Type de variable

  • number : les nombres qu'ils soitent entiers ou à virgule
  • string : représente les chaînes de caractère
  • boolean : représente les valeurs booléennes c-a-d true ou false
  • array: représente une tableau contenant des éléments du même type
  • object : représente tout type d'objet
  • null : représent la valeur "null"
  • undifined : représent la valeur "undefined"

Type de varaiable TS

  • any : représente un type non spécificié du dynamique // on peut pas l'utiliser dans nos code
  • unknown : représente un élément dont on ne connaît pas le type // on peut pas l'utiliser dans nos code
  • enum : représente un ensemble de valeur nommées
  • tuple : représente un tableau avec un nombre fixe d'élements de types différents

Les classes abstraite

Les Classes abstraites est une concept de la POO. C'est une classe qui ne peut être instancier directement , c-a-d qu'on ne peut pas créer d'objets directement à partir de de cette classe; Au lieu de cela, elle est conçu pour être utilisée comme un classe à partir de laquelle les autres peuvent hériter.

En TypeScript lorqu'une classe implémente une interface, elle doit fournir une implémentation pour toutes les propriés et méthodes déclarées dans l'interface
